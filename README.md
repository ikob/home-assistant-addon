# Home Assistant Add-ons

This repository provides Home Assistant add-ons maintained by Katsushi
Kobayashi.

## Installation

Add the following URL as an add-on repository in Home Assistant:

https://github.com/ikob/home-assistant-addon

[![Add repository to Home Assistant](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fikob%2Fhome-assistant-addon)

## Add-ons

### SINDAN Client

Continuously measures network conditions from the end user's perspective using
the SINDAN system.

See [SINDAN Client documentation](./sindan-client/README.md).

### AP Intercom Agent

Provides SIP registration, message handling, and incoming-call control for
compatible apartment intercom systems.

See [AP Intercom Agent documentation](./AP_intercom_agent/README.md).
