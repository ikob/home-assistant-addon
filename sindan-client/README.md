# Home Assistant Add-on: SINDAN Client

![Supports aarch64 Architecture][aarch64-shield]
![Supports amd64 Architecture][amd64-shield]

## About

The [SINDAN Project](https://www.sindan-net.com) provides a system that
facilitates network operations by continuously measuring network conditions
from the end user's perspective.

This add-on deploys SINDAN Client on Home Assistant and continuously measures
network conditions from the Home Assistant host.
